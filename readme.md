# Projeto Django - Loja

## 📌 Descrição
Projeto desenvolvido utilizando Django para atividade acadêmica.

## 🚀 Tecnologias utilizadas
- Python
- Django

## ▶️ Como executar o projeto

1. Clone o repositório:
git clone https://github.com/ketellylopes-bot/Loja.git

2. Entre na pasta:
cd Loja

3. Execute o servidor:
python manage.py runserver
